MISC
====

Miscellaneous tools. Enjoy.

### Install some packages required for build

* Darwin 16.0.0+ (you need a [brew](http://brew.sh/))

        brew install bsdmake

### Build and install

* FreeBSD 11+

        make 

* Darwin 16.0.0+

        bsdmake
